# Naradmuni - AI Wisdom Through SMS

## Overview
Naradmuni is a static landing page for an SMS-based AI service that provides access to OpenAI's powerful AI through simple text messages. The name is inspired by the celestial sage Narad Muni from Hindu mythology, who traveled the cosmos spreading divine wisdom.

## Project Structure
This is a simple static HTML/CSS website with the following structure:
- `index.html` - Main landing page with hero section, features, pricing, and how-it-works sections
- `beta.html` - Beta program signup page with instructions on how to join
- `css/style.css` - Complete stylesheet with futuristic design system
- `images/` - Directory containing background images and assets
- `server.py` - Python HTTP server for local development (serves on port 5000)

## Technology Stack
- Pure HTML5 and CSS3 (no frameworks)
- Python 3.11 for local development server
- Responsive design with mobile-first approach
- CSS custom properties (variables) for design system
- Intersection Observer API for scroll animations

## Development
The site is served using a simple Python HTTP server configured to:
- Run on `0.0.0.0:5000` for Replit compatibility
- Disable caching to ensure fresh content delivery
- Serve all static files from the root directory

## Deployment
Configured for static deployment on Replit:
- Deployment type: `static`
- Public directory: `.` (root)
- No build process required (pure HTML/CSS)

## Design Features
- Futuristic glassmorphism UI with gradient accents
- Animated background with pulsing radial gradients
- Smooth scroll animations using Intersection Observer
- Mobile-responsive grid layouts
- Custom typography using Google Fonts (Inter & Orbitron)
- Hover effects with glowing shadows and transforms
